# Advent of Code

This repository contains my solutions for [Advent of Code](https://adventofcode.com/) challenges.

## Structure

The repository is organized by Year, Day, and Language:

```text
/
├── YYYY/
│   ├── DD/
│   │   ├── language/
│   │   │   └── solution files
│   │   └── input.txt
└── inputs/
```

## Usage

Navigate to the specific day and language directory to run the solution.

Example:
```bash
cd 2025/01/python
python solution.py
```
